# Amazon IAM API

Amazon Identity and Access Management (IAM) enables you to manage access to AWS services and resources securely. Using IAM, you can create and manage AWS users and groups, and use permissions to allow and deny their access to AWS resources. IAM is a feature of your AWS account offered at no additional charge.

## APIs

- **Amazon IAM API** - Amazon Identity and Access Management (IAM) enables you to manage access to AWS services and resources securely.
